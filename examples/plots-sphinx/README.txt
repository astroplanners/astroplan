Astroplan Plotting Examples
***************************

Introduction
============

This gallery includes examples for using the various plotting functions
in `astroplan`.  You can also download the example IPython (Jupyter) notebooks,
located in `astroplan/examples`.

There are currently three basic types of plots you can make with `astroplan`:

* Airmass vs time
* Parallactic angle vs time
* Sky plot
